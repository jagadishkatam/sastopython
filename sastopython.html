else weather = "Cold";

# Define a function to implement if-elif-else logic
def calculate_merit(x):
    if x > 20:
        return "Warm"
    elif x > 10:
        return "Mild"
    else:
        return "Cold"

# Approach 1
old_data["weather"] = old_data["temp"].apply(calculate_merit)

# Approach 2
old_data["weather"] = old_data["temp"].apply(lambda x: calculate_merit(x))

new_data = old_data

Combining datasets

data new_data ;
set data_1 data_2 ;
run;
data_1 = pd.DataFrame(data1)
data_2 = pd.DataFrame(data2)

# Concatenate vertically (row-wise)
new_data = pd.concat([data_1,data_2], axis=0, ignore_index=True)

axis=0: Stacks data row-wise (default).
ignore_index=True: Resets the index, creating a continuous index across the rows.

data new_data ;
merge data_1 (in= in_1) data_2 ;
by id ;
if in_1 ;
run;
data_1 = pd.DataFrame(data1)
data_2 = pd.DataFrame(data2)

new_data = pd.merge(data_1, data_2, on="id", how="left", indicator=True)

indicator=True: argument in pandas.merge() adds a special column called _merge to the resulting DataFrame. This column indicates the source of each row in the merged DataFrame. Possible Values in _merge Column left_only, right_only, both.

String Manipulation

data new_data;
set old_data;

run
old_data = pd.DataFrame(data)

  • If you omit na=False, rows with NaN in job_title will raise an error.

data new_data;
set old_data;
address = tranwrd( address , "Street" , "St" );
run;
old_data = pd.DataFrame(data)
old_data["address"] = old_data["address"].str.replace("Street", "St",case=False, regex=False)
new_data = old_data
  • Set regex=False to ensure it treats the input as plain text, not a regular expression.
data new_data;
set old_data;
house_number = compress( address , , "dk" );
run;
old_data = pd.DataFrame(data)
old_data["house_number"] = old_data["address"].str.extract(r"(\d+)", expand=False)
new_data = old_data
  • r"(\d+)" In Python, prefixing a string with r (e.g., r”some_string”) tells Python to treat the string as a raw string.This means that escape characters (like \) are treated literally and not as escape sequences. For example, \n would be interpreted as two characters \ and n, rather than as a newline.
  • expand=False ensures that the result is a single pandas Series.
